デバイスプラグインについて

各デバイスにアクセスするために必要になるのがデバイスプラグインです。

デバイスプラグインは、Google PlayやGitHub上に公開されているapkをインストールすることで追加することができます。

デバイスプラグイン管理ボタンを押下することで、デバイスプラグイン一覧へ遷移します。
インストールされているデバイスプラグインを確認することができます。

全デバイスプラグインを再起動を押下することで、インストールされている全てのデバイスプラグインに対して再起動命令を送信します。
各デバイスプラグインで再起動処理が行われますので、状態が不安定になった場合などはこのボタンを押下してみてください。

デバイスプラグイン一覧

インストールされているデバイスプラグインをリストに表示します。

各リストを押下することで、デバイスプラグイン情報画面へ遷移します。

デバイスプラグイン情報

デバイスプラグインの再起動、アンイストール、設定画面の起動を行うことができます。

また、デバイスプラグインで使用出来るプロファイルの一覧なども確認することができます。
